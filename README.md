# 🏛️ Aasiriyar ELT Agent

**Aasiriyar** (Tamil: ஆசிரியர் - "Teacher") is an autonomous AI-driven language learning platform designed to take students from zero to Tamil fluency within 12 months.

## 🚀 Vision
The project is built on the **Macro-Module** learning philosophy: grouping phonetic and grammatical concepts into logical sets rather than fragmented sentences. This ensures structural mastery alongside conversational intuition.

## 🧠 Autonomous Governance
This repository is managed by an **AI Executive Leadership Team (ELT)** running within OpenClaw:
- **CEO (Vision):** Sets weekly velocity targets and ensures 12-month mastery goals.
- **CTO (Intelligence):** Manages the learning engine, curriculum logic, and TTS validation.
- **COO (Operations):** Oversees daily delivery to Discord and infrastructure stability.

### Continuous Improvement
The agent performs **Daily PR Optimizations**:
1. Analyzes the current dashboard (`index.html`) and student data (`aasiriyar-data.json`).
2. Proposes meaningful UI/UX or curriculum enhancements.
3. Performs automated code reviews and self-merges validated improvements.

## 📊 Project Components
- `index.html`: The live student dashboard (Progress, XP, and Rank).
- `aasiriyar-data.json`: The source of truth for student history and curriculum state.
- `aasiriyar-curriculum-tree.md`: The 12-month roadmap.

## 📅 Status
- **Current Phase:** Phase 1 (Foundations)
- **Active Module:** Module 2 (The Strong Sounds - Vallinam)
- **Hosting:** Cloudflare Tunnel (Live)
